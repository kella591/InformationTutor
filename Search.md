# 深度优先搜索

## 框架

```
procedure dfs(x);
  var
  begin
     if 达到目标状态 then 输出结果并退出过程;
     if 满足剪枝条件 then exit;
     for i:=1 to 搜索宽度 do 
       if 满足搜索条件 then
       begin
         备份现场；（注意如果现场使用了全局变量，则需要使用局部变量备份）
         dfs(参数+增量)；
         恢复现场；
       end
end;
```

## *优化
* 最优化剪枝：求最优值时，当前的状态无论如何不可能比最优值更优，则退出，可与展望结合剪枝
* 可行性剪枝：提前判断该状态是否能得到可行解，如不能则退出
* 记忆化搜索：对于已经搜索过的状态直接退出
* 改变搜索顺序：对于看起来希望更大的决策先进行搜索
